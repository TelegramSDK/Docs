# Announcing BotKit

[BotKit](/docs/kit) is just an extension for the [BotAPI](/docs/intro) package useful for having suggestions by your IDE.

This new package fetches periodically all the methods from the [official documentation](https://core.telegram.org/bots/api) to create all the PHP methods.
